Con este proyecto pudimos desarrollar una aplicación web con la que podemos obtener información muy útil tomando como base de datos una lista de modelos de automóviles con la cual podemos conocer como están distribuidos de acuerdo al kilometraje que tienen así como la relación de precio y kilometraje recorrido, de esta forma podemos observar los modelos con más o menos kilometraje así como el precio que tienen en el mercado considerando esta variable.
